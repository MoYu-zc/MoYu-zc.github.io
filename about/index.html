# 项目经历

  • 人员管理系统
    • 个人职责：项目后端编写、项目测试
    • 项目描述：基于Spring Boot+Vue+Spring Security构建，包括人员功能、工资功能、发送邮件模块，包含登录验证、人员信息管理、发送入职邮件、工资套账的管理、身份权限管理等功能；该项目使用了Redis存入员工信息、权限信息，可以支持大量的查询，还是用了RabbitMQ进行模块间的信息发送并手动确认是否消费信息，定时重发，确保消息可以成功传输。
    • Github：https://github.com/MoYu-zc/cloud-personnel_manage
  • 电影后台系统
    • 个人职责：项目前后端编写、数据库设计、项目测试
    • 项目描述：基于SSM构建，包括电影的上传下载以及在线观看、新闻、电影以及标签信息管理、修改个人信息等功能；该项目使用了前端JS检错机制和后端JSR303参数校验，能把绝大部分的问题都包括其中。项目完成之后，进行了学院内发布，使用人次约170人，学院老师使用该项目进行集体观影，反响不错。
    • Github：https://github.com/MoYu-zc/Film_manage
  • 共享笔记本
    • 个人职责：项目前后端编写、数据库设计、项目测试
    • 项目描述：基于SSH构建，包括个人权限管理、标签创建、删除、屏蔽，笔记信息新增、审核、删除，修改个人信息等功能；项目分为前台与后台两部分，在前台，用户可以在其中进行笔记以及文件的保存，可以选择共享或者私密，共享的笔记会展现在共享页面进行展示，而后台用来管理审核用户、笔记、文件信息；完成该项目后，进行了发布测试，至今寝室内也在使用该系统。
    • Github：https://github.com/MoYu-zc/Notes
