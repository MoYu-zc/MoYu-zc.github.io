秋招记录总结---加油，陌生人

# 秋招记录总结---加油，陌生人

首先，自我介绍：

关键词：郑州、双非渣本、男生、22届、Java后端

很幸运，可以在秋招期间，获得了 阿里、快手、滴滴 三个offer

对比很多大佬来说，我的offer可能不多，或者部门不够核心、不够好，但是个人来说我很满足，仅此而已。

而下面，也算是我的小记录吧~~

从今年3月份开始找实习，因为自己学校不是太好，了解了以往的校友进入的企业发现，进中厂已经是寥寥无几了，并且发现原来找工作还需要背八股，怀着 好奇+不自信 的心态，一个小菜鸡踉踉跄跄进入了自己的找工作之路。

春招实习

说来也可笑，因为自己的小不服（双非为什么不能进大厂），我第一次实习面试的公司竟然是阿里巴巴，当然......被虐了，一个没怎么准备的小孩被社会好好的毒打了一顿，也导致我心里阴影很大，大厂不敢想了。

之后我开始背了很多八股，开始投一些小公司的实习，boss、牛客上一直海投，很幸运进入了一家上海的小公司，因为疫情原因我进行了该公司的远程实习，我边实习边学习八股，简历上也增加了一段实习经历。在这个期间，我也通过了一些小公司的面试（很清楚的记得，第一个通过的公司是群硕，一个偏外企的小公司，当时上着课接到电话，超级开心），当时的我已经很满足了，也达到了我们学校的一个平均水平吧，心里虽然有点不服气，但是也没办法，可能我没什么机会，可能我能力就这么高。

这时候，突然有了别的机会！ 好未来 （ 当时我心中一个特别好的机会，薪资也很棒，双减政策前也算一个大厂 ），时长近一个月，终于通过了，很开心，觉得自己也不错了，如果能转正秋招躺平，然后教育行业爆炸.......，怀着忐忑的心情开始重新寻找工作机会，正巧碰上了滴滴秋储扩招实习生，我也试着投了投，心里其实下意识觉得这些大厂和自己没啥关系，没想到迎来了滴滴的面试，2面技术面，通过了滴滴秋储招聘。

我当时觉得，竟然真的通过了，滴滴也是大厂了，当时的我并不觉得我秋招能找到更好的公司，所以我当时经常说 ：“运气，运气.....可能我出去滴滴，就再也进不来”。

春招实习，就这样结束了，也算圆满，最终我去了滴滴。

秋招奋斗

一个人来到北京了，一个人租了房子，实习开始了。

我其实也没打算进行秋招，想着各种条件之下，能滴滴转正的话也就是极限了，所以我在里面也没想很多别的东西，静静等待转正时机。

直到我听说，滴滴转正也需要现场面试，而且不是一定可以转正成功的；开始紧张了，当时已经8月份中下旬了，很多公司的提前批也差不多准备结束了，我赶着这个结尾投了很多公司，当时很急切，能见到的提前批都投；结果也是让我很崩溃的，没有任何的回声。

我试探着通过内推投了阿里的正式批，为了不影响正常工作时间，每一面都是约的晚上8-9点，过程也很顺利，基本上每一面面评都不错，第二天就约了下一面，一共4+1面，想着秋招要结束了，很舒服。但是万万没想到，最后hrg面挂了...... 知道这个消息我很崩溃，尤其是了解到可能是学历学校原因，当时真的很无力；为了面阿里 ，已经到9月头了，可能很多公司hc都很少了，但是我的秋招还没开始，真的感觉那段时间很黑暗。

金九银十......明显就变成了金八银九，当时还有不到一个月时间了，需要边实习边面试，所以也不敢海投，怕笔试没时间写；就分批投了美团、字节、腾讯、快手、网易，其他的都没投了；

腾讯没消息、快手没消息、字节笔试之后没消息、美团和网易笔试之后顺利进行面试。心里想着也许好运要来了吗？然后....网易hr面挂、美团二面挂，而滴滴转正还是没有具体消息，原来真的是你觉得生活还行的时候，会给你当头一棒！

过了浑浑噩噩的几天之后，这个时候的我其实已经做好返校投投中厂的准备了，觉得自己的能力就到这儿了，有个中厂也还行了，身边人都是这样的呀。

突然有了还算好点的消息，阿里捞人，重新笔试、重新面试半放弃式的接收了再一次的邀请。有人告诉我阿里卡学历学校别试了，也有人说kpi放弃吧；并没有我抱什么希望，也许想争一口气，也许想证明自己，我是双非，但是不代表我一直就这样了。

我笔试了，一轮一轮的面试也在进行中，阿里的面试官（部门leader）真的很棒，中间还帮我催催进度，告诉我是不是通过，好好准备。真的，你们相信：**当你经历过一些后，世界都在帮你。**阿里进行到hrg面试的时候，快手的面试也来了（这个是真的没想到的，快手听说特别容易刷简历），而滴滴的转正面试也约好了时间。之后的一切都有条不紊的进行着.......

阿里前前后后一共 2次笔试，6次技术面，2次hrg面；快手3次技术面，1次hr面；滴滴转正2次技术面；当我写这个总结的时候，阿里和快手意向已经到了，滴滴oc等意向ing~~(到了会补充上)。

我在最难过、无力的时候，看到过一个师兄的文章：没有面试，没有 offer，不代表能力不行。

我这里列出来你至今 0 offer 的几个原因:

  • 有的公司没有诚意，随意出题
  • 面试太紧张了，本来可以发挥得更好
  • 自己不够重视，迟到或者忘记参加笔面试
  • 面试官安排错了或者面试官爽约
  • 岗位只有一个，其他候选人比自己优秀
  • ...

别看这几个不足挂齿的原因，在求职过程中随时可能会发生在自己身上，导致 0 offer，但你要知道这些情况都不能说明自己能力不行。

你可能成绩拔尖，俗称考霸，绩点甩同班同学几条街，但职场考的不是一本书的知识点，考的是工作能力，面试官出的考题有很强的不确定性。你可能是班上的班长，拥有很强的组织能力，但面试不会给你一群人组织（除了群面），技术岗基本上都是单独面试。你可能是acm金牌选手，但是语言表达能力欠缺，和 HR 沟通薪资的时候被挂。

坚持一下，工作会有的，面包会有的。

我是足够幸运的，度过了最黑暗的时候、度过了最孤独的时候、也度过了最无力无奈的时候。我的秋招顺利结束了，我就是普普通通的一个双非学生，经过了自己的努力，有能力也有运气更有机遇，滚起了雪球，拿了大厂offer，勉勉强强也算是 《双非如何进入大厂》，哈哈哈哈。

  • 别想着努力了就有收获，因为本来就很强的人，他们也在一直努力
  • 你的offer会来的，等一等，他可能来的有点晚
  • 尊重每一次面试，认真对待它，说不准你真的可以进去
  • 没有什么不可能、也没有什么本就应该，双非也可以进入最好的公司

这篇文章是为了总结我的秋招，也是为了给还没上岸的朋友、还处在黑暗阶段的朋友一些鼓励，加油陌生人！
